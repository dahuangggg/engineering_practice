# Workflow Logic

这个文件用 Mermaid 描述当前系统的页面流程和后端分析闭环，方便直接放到 GitHub 或答辩材料里使用。

## 页面与数据流程

```mermaid
flowchart TD
    A[进入页面] --> B{选择数据来源}

    B -->|模拟数据测试| C[选择内置场景\n平稳日/尖峰日/波动日]
    B -->|自定义数据输入| D[上传文件\nCSV/JSON/TXT/MD/PDF]

    C --> E[生成模拟市场数据]

    D --> F{文件类型}
    F -->|CSV/JSON| G[直接读取结构化数据]
    F -->|TXT/MD/PDF| H[调用 OpenAI 提取 24 小时市场数据]

    G --> I[校验字段与 24 小时时段完整性]
    H --> I
    E --> J[点击 生成数据分析可视化]
    I --> J

    J --> K[展示输入数据概览\n负荷曲线/价格曲线/明细表]
    K --> L[调整参数\n风险偏好/规则参数/模型配置]
    L --> M[点击 开始策略分析]

    M --> N{是否启用大模型}
    N -->|否| O[规则引擎执行闭环]
    N -->|是| P[OpenAI 参与规则理解\n策略建议与复盘]
    P --> O

    O --> Q[输出分析结果页面\n指标卡/智能体卡片/图表/明细表]
```

## 智能体闭环流程

```mermaid
flowchart LR
    A[输入数据\n规则文本/市场数据/参数] --> B[感知智能体]
    B --> C[识别市场状态\n高价时段/低价时段/规则约束]
    C --> D[决策智能体]
    D --> E[生成申报方案\n移峰比例/重点时段/申报电量]
    E --> F[执行智能体]
    F --> G[模拟结算\n日前成本/实时补购/回收收入/偏差罚金]
    G --> H[反馈智能体]
    H --> I[复盘收益与偏差\n输出诊断建议]
    I --> J[优化智能体]
    J --> K[给出下一轮参数建议\n推荐移峰比例]
```

## 文本文件提取流程

```mermaid
flowchart TD
    A[上传 TXT/MD/PDF] --> B[读取文本内容]
    B --> C[构造数据提取 Prompt]
    C --> D[调用 OpenAI]
    D --> E[返回 rows JSON]
    E --> F[转换为 24 小时 DataFrame]
    F --> G{校验是否完整}
    G -->|通过| H[生成输入数据概览]
    G -->|失败| I[提示用户修正文件或重新上传]
```

## 说明

- 页面现在是两阶段流程：先看输入数据，再做策略分析。
- 自定义文件里如果是结构化数据，系统直接读取；如果是文本或 PDF，系统会先让大模型提取数据。
- 大模型主要参与：规则理解、策略建议、复盘优化。
- 确定性计算主要负责：申报电量、结算成本、偏差率和最终收益计算。
